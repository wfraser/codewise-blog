description

Main subsection:

This is just a simple text field for describing the skin. It never gets used for any other purpose.
there are no local variables

css

Main subsection:

This CSS gets included in all pages.
there are no local variables

main

Main subsection:

This is the template for all pages.
there are no local variables

The versionfooter subsection:

This is the footer for all pages.
there are no local variables

postcalendar

Main subsection:

A list with links to all the months posts have been made in.
Variable Description
${contents} CWB-generated HTML to be put in


The monthrow subsection:

A link to a month and year.
Variable Description
${url} link to display the month's posts
${monthname} name of the month
${year} year, 4 digits


The latestmonth subsection:

The month currently being shown.
Variable Description
${contents} CWB-generated HTML for the current month


The topiclink subsection:

Link to a topic
Variable Description
${url} URL link to the post
${title} post title


The nextpage subsection:

Link to the next page of topics
Variable Description
${url} link to the next page


loginform

Main subsection:

The form to login to the control panel
Variable Description
${url} URL to post the login details at


welcomeback

Main subsection:

A 'welcome back' message including the visitor's name, if known.
Variable Description
${contents} filled with a call to the name subsection if user's name is known, otherwise filled with a call to the noname subsection.
${url} the URL to delete session information


The name subsection:

Used when the visitor's name is known
Variable Description
${name} the known name of the visitor


The noname subsection:

Used when the visitor's name is not known
there are no local variables

topic

Main subsection:

For displaying blog topics
Variable Description
${text} complete processed text of the post


The topicheader subsection:

The header of the topic
Variable Description
${date} formatted timestamp for when the post was posted
${url_showtopic} URL of the post
${title} the post's title


The topicfooter subsection:

The footer of the topic
Variable Description
${num_comments} number of comments on the post
${url_showcomments} URL to show the comments
${url_addcomment} URL to display the form to add a comment


The topicfooter_addcomment subsection:

A link to display the form to add a comment
Variable Description
${url_addcomment} URL to display the form to add a comment


The start_comments subsection:

Shown before the list of comments. there are no local variables

The last_comments subsection:

Used in place of start_comments subsection when the reply form is shown
Variable Description
${num} the number of comments being shown


controlpanel

Main subsection:

The template for all control panel pages.
Variable Description
${root} true if user is root
${current} set to the name of the current controlpanel page
${cpurl} URL to the controlpanel main page
${url_settings} URL to the Settings control panel page
${url_write} URL to the Write control panel page
${url_edit} URL to the Edit control panel page
${url_userinfo} URL to the User Info control panel page
${url_skin} URL to the Skin Editor control panel page


The div subsection:

Divider between control panel page links
there are no local variables

The versionfooter subsection:

The footer for all control panel pages.
there are no local variables

replyform

Main subsection:

The form for posting a comment
Variable Description
${form_url} URL the form is to be POSTed to
${name} poster's name
${tripcode} poster's tripcode
${tripcode_help_link} link to some informative text about tripcodes
${link} poster's link
${text} text of the comment
${text_filter_msg} messages from the HTML filter
${allowed_tags} list of allowed HTML tags


pagelink

Main subsection:

Container for the page links
there are no local variables

The pagelink subsection:

Link to a page
Variable Description
${url} URL to display the page
${pageno} page number


The currpage subsection:

Static name of the current page
Variable Description
${pageno} page number


The allonepage_link subsection:

Link to display everything on all one page
Variable Description
${url} URL to display all one page


The allonepage_current subsection:

Static text stating that everything is on one page.
there are no local variables

post

Main subsection:

A comment on a blog post.
Variable Description
${special_anchor} a special additional anchor to use for the post
${pid} post ID
${highlight} True if the post should be highlighted
${name} name of the poster
${tripcode} poster's tripcode
${date} date posted
${url_post} URL of the specific post
${url_reply} URL to open the reply form with a reply to the current post
${text} text of the post


subscribeform

Main subsection:

The form for subscribing to a blog
Variable Description
${url} URL the form is POSTed to


error

Main subsection:

Nothing; no purpose
there are no local variables

The error subsection:

For displaying an error
Variable Description
${message} message text


The notify subsection:

For notifying the user about something (not an error)
Variable Description
${message} message text


shoutbox

Main subsection:

The whole shoutbox
Variable Description
${contents} all the shoutbox posts
${posturl} URL the form is POSTed to
${name} poster's name
${link} poster's link


The row_even subsection:

Even numbered rows
Variable Description
${link} poster's link
${name} poster's name
${text} text of the post
${date} date posted


The row_odd subsection:

Odd numbered rows
same as the row_even subsection

The nothing subsection:

Used when there are no shouts to display
there are no local variables

controlpanel_write

Main subsection:

The Write page of the control panel.
Variable Description
${posturl} URL the form is POSTed to
${autoresize} URL of the autoresize javascript file
${quicktags} URL of the quicktags javascript file
${title} title of the topic
${rows} number of rows in the text box
${cols} number of columns in the text box
${text} topic text


The success_redirect subsection:

Redirects upon a succesful post
Variable Description
${topic_url} URL of the newly posted topic


The preview_topic subsection:

Container for the preview text of the post
Variable Description
${text} text of the topic


controlpanel_settings

Main subsection:

The Settings page of the control panel. Only accessible to administrators.
Variable Description
${posturl} URL the form is POSTed to
${fspath} the value of the FSPATH config setting
${sql_admin_email} the value of the SQL_ADMIN_EMAIL config setting
${custom_url_enabled} the value of the CUSTOM_URL_ENABLED config setting
${email} the value of the EMAIL config setting
${sql_host} the value of the SQL_HOST config setting
${sql_user} the value of the SQL_USER config setting
${sql_pass} the value of the SQL_PASS config setting
${sql_db} the value of the SQL_DB config setting
${sample_url} sample user's URL using the given settings
${main_url} the main CodewiseBlog URL using the given settings
${subdomain_mode} the value of the SUBDOMAIN_MODE config setting
${base_domain} the value of the BASE_DOMAIN config setting
${installed_path} the value of the INSTALLED_PATH config setting
${default_subdomain} the value of the DEFAULT_SUBDOMAIN config setting
${topics_per_page} the value of the TOPICS_PER_PAGE config setting
${posts_per_page} the value of the POSTS_PER_PAGE config setting
${shouts_per_page} the value of the SHOUTS_PER_PAGE config setting
${date_format} the value of the DATE_FORMAT config setting
${anonymous_name} the value of the ANONYMOUS_NAME config setting
${allowed_tags} the value of the $ALLOWED_TAGS config setting


controlpanel_edit

Main subsection:

The Edit page of the control panel
there are no local variables

The showselect subsection:

A <select> tag with a list of topics to edit
Variable Description
${posturl} URL the form is POSTed to
${options} list of topics (for use with a <select> tag)


The editform subsection:

The form to edit the topic in
Variable Description
${posturl} URL the form is POSTed to
${title} title of the topic
${month_sel} the <select> tag for choosing the month
${date_sel} the <select> tag for choosing the date
${year_sel} the <select> tag for choosing the year
${hour_sel} the <select> tag for choosing the hour
${minute_sel} the <select> tag for choosing the minute
${second_sel} the <select> tag for choosing the second
${ampm_sel} the <select> tag for choosing between AM and PM
${preview} fully formatted post text preview
${autoresize} URL of the autoresize javascript file
${quicktags} URL of the quicktags javascript file
${rows} number of rows for the textbox
${cols} number of columns for the textbox
${text} the text of the topic
${tid} topic ID


The success_redirect subsection:

For redirecting upon a successful edit
Variable Description
${topic_url} URL for the edited topic


The delete_ask subsection:

Asks for confirmation for deleting a post
Variable Description
${posturl} URL the form is POSTed to
${tid} topic ID


The delete_successful subsection:

Indicates that the delete was successful
Variable Description
${num_comments} number of comments the topic previously had


controlpanel_userinfo

Main subsection:

The User Info page of the control panel
Variable Description
${posturl} URL the form is POSTed to
${email} user's email address
${realname} user's real name
${birthday} user's birthday
${location} user's location
${interests} user's interests (may contain HTML)
${links} user's links (may contain HTML)
${photo} URL of the user's photo or avatar
${homepage} URL of the user's homepage
${title} title of the user's blog
${custom_url} custom URL the blog appears at


controlpanel_skin

Main subsection:

The Skin Editor page of the control panel
Variable Description
${posturl} URL the form is POSTed to
${sectionlist} the list of skin sections as a row of buttons
${varlist} an <iframe> with this document in it, scrolled to the appropriate section
${content} the skin section's text (HTML escaped)
${section_name} name of the currently viewed skin section. FALSE if none is being viewed.


The sectionlist_entry subsection:

A button in the list of skin sections
Variable Description
${section} name of the skin section


The sectionlist_current subsection:

The currently viewed section
Variable Description
${section} name of the skin section


The section_edit subsection:

Contains the textbox and other items for editing a skin section
Variable Description
${autoresize} URL of the autoresize javascript file
${section_name} name of the section
${using_master} TRUE if the master skin is being used for this section
${rows} number of rows in the textbox
${cols} number of columns in the textbox
${section_content} full text of the skin section (HTML escaped)


controlpanel_adduser

Main subsection:

The Add User page of the control panel. Only accessible to administrators.
Variable Description
${posturl} The URL the form gets POSTed to
${blogid} The Blog ID the new member will have


controlpanel_skin_multi

Main subsection:

The Skin Editor page of the control panel. Shown once a skin has been selected from the skin_select subsection.
Variable Description
${posturl} The URL the form gets POSTed to
${skinid} The Skin ID of the currently shown skin
${sectionlist} the list of skin sections as a row of buttons
${content} the skin section's text (HTML escaped)
${section_name} name of the currently viewed skin section. FALSE if none is being viewed.
${varlist} an <iframe> with this document in it, scrolled to the appropriate section


The sectionlist_entry subsection

An entry in the list of sections.
Variable Description
${section} the name of the section


The sectionlist_current subsection

The entry in the list of sections for the currently viewed section.
Variable Description
${section} the name of the section


The section_edit subsection

Contains the text box for editing the skin section, etc.
Variable Description
${autoresize} URL of the autoresize javascript file
${section_name} name of the section being edited
${using_master} TRUE if the master skin is being used for this section
${rows} number of rows in the textbox
${cols} number of columns in the textbox
${section_content} full text of the skin section (HTML escaped)


The skin_select subsection

Lets user select a saved skin for editing
Variable Description
${posturl} the URL the form is POSTed to
${savedskins} CWB-generated list of saved skins


The saved_skinids_entry subsection

An entry in the list of saved skins
Variable Description
${skinid} the Skin ID to select
${name} the name of the corresponding skin


The saved_skinids_current subsection

The currently used skin in the list of saved skins.
same as saved_skinids_entry

The skin_name subsection

Allows user to edit the skin name when the description page is shown
Variable Description
${name} The current name of the skin


The skin_owner subsection

Allows administrators to change the skin's owner when the description page is shown
Variable Description
${owner} the Blog ID of the skin's current owner
