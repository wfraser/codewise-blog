css

Main subsection:
there are no local variables

main

Main subsection:
there are no local variables

For the versionfooter subsection:
there are no local variables

postcalendar

Main subsection:
Variable Description
${contents} CWB-generated HTML to be put in


For the monthrow subsection:
Variable Description
${url} link to display the month's posts
${monthname} name of the month
${year} year, 4 digits


For the latestmonth subsection:
Variable Description
${contents} CWB-generated HTML for the current month


For the topiclink subsection:
Variable Description
${url} URL link to the post
${title} post title


For the nextpage subsection:
Variable Description
${url} link to the next page


loginform

Main subsection:
Variable Description
${url} URL to post the login details at


welcomeback

Main subsection:
Variable Description
${contents} filled with a call to the name subsection if user's name is known, otherwise filled with a call to the noname subsection.
${url} the URL to delete session information


For the name subsection:
Variable Description
${name} the known name of the visitor


For the noname subsection:
there are no local variables

topic

Main subsection:
Variable Description
${text} complete processed text of the post


For the topicheader subsection:
Variable Description
${date} formatted timestamp for when the post was posted
${url_showtopic} URL of the post
${title} the post's title


For the topicfooter subsection:
Variable Description
${num_comments} number of comments on the post
${url_showcomments} URL to show the comments
${url_addcomment} URL to display the form to add a comment


For the topicfooter_addcomment subsection:
Variable Description
${url_addcomment} URL to display the form to add a comment


For the start_comments subsection:
there are no local variables

For the last_comments subsection:
Variable Description
${num} the number of comments being displayed at a time


controlpanel

Main subsection:
Variable Description
${root} true if user is root
${current} set to the name of the current controlpanel page
${cpurl} URL to the controlpanel main page
${url_settings} URL to the Settings control panel page
${url_write} URL to the Write control panel page
${url_edit} URL to the Edit control panel page
${url_userinfo} URL to the User Info control panel page
${url_skin} URL to the Skin Editor control panel page


For the div subsection:
there are no local variables

For the versionfooter subsection:
there are no local variables

replyform

Main subsection:
Variable Description
${form_url} URL the form is to be POSTed to
${name} poster's name
${tripcode} poster's tripcode
${tripcode_help_link} link to some informative text about tripcodes
${link} poster's link
${text} text of the comment
${text_filter_msg} messages from the HTML filter
${allowed_tags} list of allowed HTML tags


pagelink

Main subsection:
there are no local variables

For the pagelink subsection:
Variable Description
${url} URL to display the page
${pageno} page number


For the currpage subsection:
Variable Description
${pageno} page number


For the allonepage_link subsection:
Variable Description
${url} URL to display all one page


For the allonepage_current subsection:
there are no local variables

post

Main subsection:
Variable Description
${special_anchor} a special additional anchor to use for the post
${pid} post ID
${highlight} True if the post should be highlighted
${name} name of the poster
${tripcode} poster's tripcode
${date} date posted
${url_post} URL of the specific post
${url_reply} URL to open the reply form with a reply to the current post
${text} text of the post


subscribeform

Main subsection:
Variable Description
${url} URL the form is POSTed to


error

Main subsection:
there are no local variables

For the error subsection:
Variable Description
${message} message text


For the notify subsection:
Variable Description
${message} message text


shoutbox

Main subsection:
Variable Description
${contents} all the shoutbox posts
${posturl} URL the form is POSTed to
${name} poster's name
${link} poster's link


For the row_even subsection:
Variable Description
${link} poster's link
${name} poster's name
${text} text of the post
${date} date posted


For the row_odd subsection:
same as the row_even subsection

For the nothing subsection:
there are no local variables

controlpanel_write

Main subsection:
Variable Description
${posturl} URL the form is POSTed to
${autoresize} URL of the autoresize javascript file
${quicktags} URL of the quicktags javascript file
${title} title of the topic
${rows} number of rows in the text box
${cols} number of columns in the text box
${text} topic text


For the success_redirect subsection:
Variable Description
${topic_url} URL of the newly posted topic


For the preview_topic subsection:
Variable Description
${text} text of the topic


controlpanel_settings

Main subsection:
Variable Description
${posturl} URL the form is POSTed to
${fspath} the value of the FSPATH config setting
${sql_admin_email} the value of the SQL_ADMIN_EMAIL config setting
${custom_url_enabled} the value of the CUSTOM_URL_ENABLED config setting
${email} the value of the EMAIL config setting
${sql_host} the value of the SQL_HOST config setting
${sql_user} the value of the SQL_USER config setting
${sql_pass} the value of the SQL_PASS config setting
${sql_db} the value of the SQL_DB config setting
${sample_url} sample user's URL using the given settings
${main_url} the main CodewiseBlog URL using the given settings
${subdomain_mode} the value of the SUBDOMAIN_MODE config setting
${base_domain} the value of the BASE_DOMAIN config setting
${installed_path} the value of the INSTALLED_PATH config setting
${default_subdomain} the value of the DEFAULT_SUBDOMAIN config setting
${topics_per_page} the value of the TOPICS_PER_PAGE config setting
${posts_per_page} the value of the POSTS_PER_PAGE config setting
${shouts_per_page} the value of the SHOUTS_PER_PAGE config setting
${date_format} the value of the DATE_FORMAT config setting
${anonymous_name} the value of the ANONYMOUS_NAME config setting
${allowed_tags} the value of the $ALLOWED_TAGS config setting


controlpanel_edit

Main subsection:
there are no local variables

For the showselect subsection:
Variable Description
${posturl} URL the form is POSTed to
${options} list of topics (for use with a <select> tag)


For the editform subsection:
Variable Description
${posturl} URL the form is POSTed to
${title} title of the topic
${month_sel} the <select> tag for choosing the month
${date_sel} the <select> tag for choosing the date
${year_sel} the <select> tag for choosing the year
${hour_sel} the <select> tag for choosing the hour
${minute_sel} the <select> tag for choosing the minute
${second_sel} the <select> tag for choosing the second
${ampm_sel} the <select> tag for choosing between AM and PM
${preview} fully formatted post text preview
${autoresize} URL of the autoresize javascript file
${quicktags} URL of the quicktags javascript file
${rows} number of rows for the textbox
${cols} number of columns for the textbox
${text} the text of the topic
${tid} topic ID


For the success_redirect subsection:
Variable Description
${topic_url} URL for the edited topic


For the delete_ask subsection:
Variable Description
${posturl} URL the form is POSTed to
${tid} topic ID


For the delete_successful subsection:
Variable Description
${num_comments} number of comments the topic previously had


controlpanel_userinfo

Main subsection:
Variable Description
${posturl} URL the form is POSTed to
${email} user's email address
${realname} user's real name
${birthday} user's birthday
${location} user's location
${interests} user's interests (may contain HTML)
${links} user's links (may contain HTML)
${photo} URL of the user's photo or avatar
${homepage} URL of the user's homepage
${title} title of the user's blog
${custom_url} custom URL the blog appears at


controlpanel_skin

Main subsection:
Variable Description
${posturl} URL the form is POSTed to
${sectionlist} the list of skin sections as a row of buttons
${varlist} currently, nothing (FIXME)
${content} the skin section's text (HTML escaped)
${section_name} name of the currently viewed skin section. FALSE if none is being viewed.


For the sectionlist_entry subsection:
Variable Description
${section} name of the skin section


For the sectionlist_current subsection:
Variable Description
${section} name of the skin section


For the section_edit subsection:
Variable Description
${autoresize} URL of the autoresize javascript file
${section_name} name of the section
${using_master} TRUE if the master skin is being used for this section
${rows} number of rows in the textbox
${cols} number of columns in the textbox
${section_content} full text of the skin section (HTML escaped)
